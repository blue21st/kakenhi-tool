# 科研費取得合計算出ツール

KAKEN（科学研究費助成事業データベース）の公開APIを使用して、研究者の科研費取得履歴を検索・集計するローカルツールです。

## 必要環境

- **Python 3.7以上**（標準ライブラリのみ使用、追加インストール不要）
- **インターネット接続**（KAKEN APIへのアクセス用）
- **Webブラウザ**

## セットアップ

### 1. CiNii API利用登録（推奨）

KAKEN APIを利用するには、CiNii APIのアプリケーションIDが必要です。

1. [CiNii API利用登録ページ](https://support.nii.ac.jp/ja/cinii/api/developer) にアクセス
2. フォームに記入して登録
3. メールでアプリケーションID（appid）が届く

> **注意**: appidなしでも検索は試行されますが、レスポンスが制限される場合があります。

### 2. サーバー起動

```bash
cd kakenhi-tool
python server.py
```

以下のメッセージが表示されれば起動成功です：

```
============================================================
  科研費取得合計算出ツール
  サーバー起動: http://localhost:8080
  終了するには Ctrl+C を押してください
============================================================
```

### 3. ブラウザでアクセス

ブラウザで以下のURLを開きます：

```
http://localhost:8080
```

## 使い方

1. **研究者フルネーム**を入力（姓と名の間にスペース）
2. 必要に応じて**所属機関名**を入力（同姓同名対策）
3. 「API設定」からappidを入力（推奨）
4. **検索・集計**ボタンをクリック
5. 結果が表示されます：
   - 科研費合計額
   - 総課題数・金額取得可能課題数
   - 課題一覧テーブル（ソート可能）
6. **CSV出力**ボタンでデータをダウンロード可能

## 制約・注意事項

- 本ツールはKAKEN公開情報に基づく集計です
- 金額未公開の課題は合計に含まれません
- 同姓同名の研究者の課題が含まれる可能性があります
- 合計額は参考値であり、正式な金額を保証しません
- 古い課題ほど金額情報が欠損している傾向があります

## 技術仕様

- **フロントエンド**: HTML + CSS + JavaScript（フレームワーク不使用）
- **バックエンド**: Python 標準ライブラリ（http.server, urllib, xml.etree）
- **API**: KAKEN OpenSearch API
- **通信**: バックエンドがKAKEN APIにリクエスト → XMLパース → JSONでフロントエンドに返却
